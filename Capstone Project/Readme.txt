# Capstone Project – LearnSimple  
A beginner-friendly educational landing page.

## Project Theme  
**LearnSimple** is a simple learning platform landing page.  
It introduces web development courses with a clean layout, easy navigation, and a responsive design.

## Sections Implemented  

### **1. Header**
- Logo & brand name  
- Navigation bar  
- Simple, beginner-friendly layout  

### **2. Hero Section**
- Heading & short description  
- Two buttons (Explore Courses & See Pricing)  
- Feature badges  
- Hero image  

### **3. Courses Section**
- Three cards  
- Each card contains:  
  - Image  
  - Title  
  - Short description  
  - Start button  

### **4. Testimonials Section**
- Rating box  
- Two student feedback cards  
- Responsive layout  

### **5. Pricing Section**
- Starter & Pro plans  
- Pricing details + Choose button  

### **6. Contact Section**
- Contact form with:  
  - Full name field  
  - Email field  
  - Course dropdown  
  - Message textarea  

## Main Layout Techniques  

###  Flexbox
Used for:
- Header  
- Footer  
- Contact layout  
- Buttons  

### CSS Grid
Used for:
- Hero section  
- Courses cards  
- Testimonials layout  
- Pricing cards  

### Responsive Images
Implemented using:
- `object-fit: cover;`
- `max-width: 100%;`
- Rounded corners  

## Breakpoints Used (Responsiveness)

### **1. max-width: 980px**
- Hero shrinks  
- Testimonials adjust  
- Pricing becomes one column  

### **2. max-width: 760px**
- Header becomes vertical  
- Hero becomes single column  
- Cards stack into one column  
- Testimonials stack  

### **3. max-width: 420px**
- Smaller text  
- More compact layout  

##  How to Open the Project (Locally)

1. Download the project ZIP or clone from GitHub.  
2. Open the folder.  
3. Double-click **index.html**.  
4. The website will open in your browser.
